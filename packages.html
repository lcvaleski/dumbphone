Apple Configurator Guide — Sample

Here's a taste of what's inside. The full bundle includes Apple Configurator, Screen Time, Blocking Apps guides, and a weekly newsletter.

1

Download Apple Configurator 2

Open the Mac App Store and search for Apple Configurator 2. It's free. You'll need a Mac — this tool is Mac-only.

2

Put the iPhone in Recovery mode

Connect the iPhone to your Mac via USB. In Apple Configurator, go to Actions → Prepare. Follow the on-screen prompts to supervise the device. This will erase it, so do this before handing it to your child.

Heads up: Supervising the device erases all existing data. Set it up from scratch or restore from a backup after supervising.
3

Create a configuration profile

In Apple Configurator, click File → New Profile. Under the Restrictions payload, disable app installation, Safari, and prevent passcode changes.

4

Install the profile on the device

With the iPhone connected, drag your .mobileconfig file onto the device in Apple Configurator. The profile installs silently.

5

Lock it with a passcode your child doesn't know

Enforce minimum complexity and set a maximum number of failed attempts from the Passcode payload.
